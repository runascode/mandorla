# Exp 06 Stage A — Reading Notes

Per-paper deep-read notes for the substrate-rethinks survey. One
§-section per paper in the deep-read set (Task 2), organized by lineage.
See [`STAGE-A-PLAN.md`](STAGE-A-PLAN.md) Tasks 2.1–2.5 for the deep-read
acceptance criteria.

The deep-read set for each lineage is the T1 papers from
[`READING-LIST.md`](READING-LIST.md) plus 1–3 T2 papers chosen to fill
specific gaps (newest, largest-scale, most-cited). Target: 15–25 deep
reads across the five lineages.

## Per-paper template

Copy and adapt for each deep-read paper:

    ### [Citation, Year]

    **Claim:**
    **Method (in one paragraph):**
    **Scale demonstrated:**
    **Axes moved:** input ☐ representation ☐ objective ☐
    **Failure mode / limitation (if any, at what scale):**
    **Why it does / does not bear on "all three together":**
    **Relation to MANDORLA (if any):**

Axes flags are filled definitively here (not provisionally as in
`READING-LIST.md`) — a deep read is the basis for the matrix row in
[`findings/substrate-rethinks-survey.md`](../../findings/substrate-rethinks-survey.md).
