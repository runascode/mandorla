# Exp 06 Stage A — Reading List

Paper shortlist for the substrate-rethinks survey, organized by the
five lineages named in [`PILOT-DESIGN.md`](PILOT-DESIGN.md) §4.1.
See [`STAGE-A-PLAN.md`](STAGE-A-PLAN.md) Task 1 for tiering and
acceptance criteria.

**Tier definitions:**

- **T1** — canonical / load-bearing for the lineage; substrate-level
  claim (replaces input, representation, or objective primitive *at
  scale*).
- **T2** — significant; extends, contests, or operationalizes a T1.
- **T3** — adjacent; cited in the lineage but not load-bearing for
  the "all three together" question.

**Axes-moved flags** are *provisional* at shortlist time and sharpened
by deep-read in Task 2. `I` = input primitive, `R` = representation
primitive, `O` = objective primitive. Use `Y` / `N` / `?`.
